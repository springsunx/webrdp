# WebRDP 输入界面使用说明

## 功能特性

### 1. 输入参数界面

当不带参数访问时，会显示美观的输入界面：

```
http://localhost:3000/
```

输入界面包含：
- **主机地址**（必填）
- **端口**（默认3389）
- **用户名**（必填）
- **密码**（必填）
- **屏幕宽度**（默认1024）
- **屏幕高度**（默认768）
- **记住连接信息**（复选框）

### 2. URL参数访问

也可以通过URL参数直接连接：

```
http://localhost:3000?host=YOUR_RDP_HOST&port=3389&user=YOUR_USER&password=YOUR_PASSWORD
```

### 3. 记住连接信息

勾选"记住连接信息"后，下次访问会自动填充上次的连接信息。

## 使用流程

### 流程1：首次访问

1. 访问 `http://localhost:3000/`
2. 输入连接信息
3. 勾选"记住连接信息"（可选）
4. 点击"连接"按钮
5. 等待连接成功

### 流程2：再次访问

1. 访问 `http://localhost:3000/`
2. 输入信息会自动填充
3. 点击"连接"按钮
4. 等待连接成功

### 流程3：URL参数访问

1. 访问 `http://localhost:3000/?host=...&user=...&password=...`
2. 自动连接，无需手动输入

## 界面功能

### 远程桌面界面

- **状态显示**: 显示当前连接状态
- **断开按钮**: 断开当前连接
- **重连按钮**: 重新连接
- **全屏按钮**: 切换全屏模式
- **返回按钮**: 返回登录界面

### 底部控制栏

- **连接信息**: 显示当前连接的主机、端口、用户
- **分辨率调整**: 可以调整屏幕分辨率

## 端口配置

| 服务 | 容器端口 | 主机端口 | 说明 |
|------|----------|----------|------|
| Guacd | 4822 | 4822 | RDP协议代理 |
| 后端API | 3001 | 3001 | REST API服务 |
| WebSocket | 3002 | 3002 | WebSocket服务 |
| 前端 | 3000 | 3000 | Web界面 |

## 快捷键

- **F11**: 切换全屏模式

## 数据存储

连接信息保存在浏览器的localStorage中：
- **Key**: `webrdp-params`
- **数据**: host, port, user, password, width, height

## 安全注意事项

1. **密码安全**: 连接信息保存在浏览器本地，仅在本机安全
2. **HTTPS**: 生产环境建议使用HTTPS
3. **访问控制**: 建议实施访问控制策略

## 故障排除

### 问题1: 无法保存连接信息

**原因**: 浏览器禁用了localStorage

**解决**: 
1. 检查浏览器设置
2. 允许网站使用localStorage

### 问题2: 自动填充不工作

**原因**: 浏览器自动填充功能被禁用

**解决**:
1. 手动输入连接信息
2. 勾选"记住连接信息"

### 问题3: 连接失败

**原因**: 网络问题或RDP服务器配置问题

**解决**:
1. 检查RDP服务器是否可达
2. 检查用户名和密码是否正确
3. 查看浏览器控制台的错误信息
